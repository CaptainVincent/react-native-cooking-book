# 建構第一個 App

透過 React-Native command line tool 創建一個 project 名稱為 **FirstProject**
```bash
react-native init FirstProject
```

產生玩的目錄結構
```
./
|-- __tests__
|   |-- index.android.js
|   `-- index.ios.js
|-- android
|   |-- app
|   |-- build.gradle
|   |-- gradle
|   |-- gradle.properties
|   |-- gradlew
|   |-- gradlew.bat
|   |-- keystores
|   `-- settings.gradle
|-- index.android.js
|-- index.ios.js
|-- ios
|   |-- FirstProject
|   |-- FirstProject.xcodeproj
|   `-- FirstProjectTests
|-- node_modules
|   |-- babel-jest
|   |-- babel-preset-react-native
|   |-- jest
|   |-- jest-react-native
|   |-- react
|   |-- react-native
|   |-- react-test-renderer
|   `-- whatwg-fetch
`-- package.json
```
